# Telegram Quote Bot
<p>   follow me in @p33_9 in telegram </p>

A Telegram bot that automatically publishes random quotes from a public API to a specified channel at regular intervals.

## Features

- Publishes a new quote every 3 seconds (adjustable) to your Telegram channel.
- Fetches quotes from the [Forismatic API](https://forismatic.com/).
- Checks if the bot has admin privileges in the target channel before starting.
- Simple command: `/start` – verifies admin status and begins publishing.

## Requirements

- Python 3.7 or higher
- A Telegram Bot Token (obtain from [@BotFather](https://t.me/BotFather))
- A Telegram channel where the bot is added as an administrator
